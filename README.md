<h1>C# - Car Sale Calculator</h1>

<h2>Descirption</h2>
This code is programmed to be able to calculate and give estimates on the discounted price, loans offered, and the price of the car. The calculations are based off whether the car is new or pre-owned, the type of car, how much the down payment is, and the users credit score.

<h2>Languages and Utilites Used</h2>

- <b>Visual Studio</b>
- <b>C#</b>

<h2>Enviornments Used</h2>
